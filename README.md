# FS-TSWB

This is the demo code for the Fixed-Support Tree-sliced Wasserstein Barycenter.
The sample code is contained in examples.ipynb.

## Requirements
- Python (3.7.4)
- POT (0.7.0)
- scipy (1.3.1)
- numpy (1.19.5)
- treelib (1.6.1)
- sklearn (0.21.3)
- pytorch (1.7.0)
- mat4py (0.4.3)
- gensim (3.8.0)
- nltk (3.4.5)
- tqdm
